# FPGA CAR

###使用FPGA與verilog做出一台可以由手機藍芽遙控，運用藍芽模組接收遙控訊號的車子，另外的模式是可以讓車子自動行徑而不靠遙控，車子運用紅外線避障模組來感測前方障礙物並且及時轉彎再前行
